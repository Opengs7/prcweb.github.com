World Migration 2010

A visualisation of world migration using animated arrows to show migration between countries. Each circle is proportional in radius to that country's emigrants (those leaving their country) or its immigrants (those settling in a new country).

Hover over a circle to show the migrant flow. The width of the arrow is proportional to the number of migrants.

This visualisation is built by Peter Cook using data from The World Bank. Built with Raphael, jQuery and Tim.
