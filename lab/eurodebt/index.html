European Debt

A visualisation of Europe's debt using animated arrows to show debt between countries. Each red circle is proportional in radius to that country's debt. Hover over a circle to show how much is owed to other countries (the width of the arrow is proportional to the amount owed).

This visualisation uses similar data to this overview from the NY Times and this chord diagram. Debt data is from here whilst the world map data is taken from this Raphael example. I do not know the original source of the debt data and cannot guarantee its veracity!

The visualisation you see here is built by Peter Cook using Raphael.js with help from the Tim micro-templater.
