3. The data join

Joins a data array with DOM elements


e.g. join an array to circle elements


4. The update (1/2)

Update attributes and style of DOM elements based on the joined data

Most commonly: position, length, radius & colour are updated based on data

4. The update (2/2)

Creation and removal of DOM elements


Enter and exit

  • an update selection is the result of a data join:
    • var u = d3.selectAll('circle').data([10, 40, 30, 20]);
  • if data has more elements than the selection, new DOM elements are added using:
    • u.enter().append('circle');
  • if data has less elements than the selection, surplus DOM elements are removed using:
    • u.exit().remove();

Enter and exit

Make selection

s = d3.select('div#chart').selectAll('div')

Join data (resulting in an 'update selection')

u = s.data(values)

Append new DOM elements

u.enter().append('div')

Remove old DOM elements

u.exit().remove()
